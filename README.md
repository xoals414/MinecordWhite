# MinecordWhite
마인크래프트 플러그인. Discord/명령을 통해 Minecraft 서버에 화이트리스트를 추가하는 플러그인 (한국어)

라이센스: MIT


---

마인코드 화이트

마인코드 화이트는 디스코드 봇으로 /명령어로 닉네임을 적으면 마인크래프트 서버에 화이스트리스트에 추가 되도록 만들어 졌습니다.
하지만 아직 개발 단계로 오류가 있을 수 있고 작업 단계가 조금 복잡합니다.
